# Security Policy

## Reporting a Vulnerability

Rendetalje.dk tager sikkerhed alvorligt. Hvis du finder en sårbarhed, bedes du kontakte os direkte:

- **Email:** info@rendetalje.dk
- **Emne:** "Sikkerhedsrapport"

Vi bestræber os på at svare inden for 48 timer.

## Scope

Følgende er in scope for sikkerhedsrapporter:

- Rendetalje.dk landing page
- API endpoints (`/api/quote`)
- Kontaktformular og datahåndtering

## Responsible Disclosure

- Del ikke sårbarheden offentligt før vi har haft mulighed for at løse den
- Vi giver statusopdatering inden for 5 hverdage
- Vi anerkender dit bidrag når problemet er løst

## Sikkerhedspraksis

Se https://rendetalje.dk/privatlivspolitik for vores databeskyttelsespraksis.
